#### Nodejs Installation
```bash
  sudo apt update
  sudo apt install nodejs
  sudo apt install npm
  nodejs -v
```
#### Clone the project
```bash
  git clone https://github.com/Alch3ms/whois.git
```
#### Go to the project directory
```bash
  cd whois
```
#### Required Installations
```bash
  npm i underscore
```
#### Run the server
```bash
  node index.js
```
#### How to Use Whois
```bash
 http://localhost/whois/${domain}
```
